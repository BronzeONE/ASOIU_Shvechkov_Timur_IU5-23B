# ИДЗ №2, вариант 26

Консольное приложение на C# для предметной области `Школы -> Ученики` с использованием:

- `SQLite` для хранения данных;
- `CSV` для начальной загрузки данных;
- `ООП` для моделей и менеджера базы;
- `Fluent Interface` в классе `ReportBuilder`.

Реализовано:

- просмотр школ и учеников;
- добавление, редактирование и удаление учеников;
- фильтр по школе;
- три обязательных отчёта;
- дополнительные функции из раздела по группам:
  - `Numbered()`;
  - `Footer(string label)`;
  - `SaveToFile(string path)`.

## Запуск

```bash
dotnet run
```

## Структура

- `School.cs`, `Student.cs` — классы-модели;
- `DatabaseManager.cs` — работа с SQLite и импорт CSV;
- `ReportBuilder.cs` — построение отчётов;
- `Program.cs` — консольное меню;
- `data/` — тестовые CSV-файлы;
- `diagrams/` — PlantUML-схемы.
